Проект на данный момент содержит: 1.)Библиотеку, с пока, что единсвенный классом City, обьекту которого можно задать имя и координаты. 2.) Форму с координатной сеткой, на которой можно выставлять обьекты класса City(в ручную- двойным щелчком и через кнопку Random, укзав количество), которые будут заноситься в список. 3.) Форму с таблицей расстояний, в которую можно импортировать матрицу из txt файла или вводить данные вручную(при ручном вводе, таблица всегда будет содержать верную расстояние, для импортирования такая проверка еще не готова). Форма открывается по нажатию "Таблица". 4.) Создание случайного маршрута(просто тест графики).

Матрицу можно добавть нажав "Add" в форма "Таблица". Там же можно вводить свои значения расстояний. Расстояние между теми городами, что указаны на карте тоже туда записывается
